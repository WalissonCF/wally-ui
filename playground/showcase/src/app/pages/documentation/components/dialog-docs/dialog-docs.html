Property Type Default Description

> Components

Component Description
Dialog Root component that provides dialog state and service
DialogTrigger Wrapper for the element that opens the dialog on click
DialogContent Container for dialog content with glass blur backdrop
DialogHeader Container for title and description with proper spacing
DialogFooter Container for action buttons with responsive layout
DialogTitle Styled heading for the dialog title
DialogDescription Styled text for the dialog description
DialogClose Close button (X) positioned in top-right corner
